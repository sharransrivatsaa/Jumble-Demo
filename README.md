# Jumble-Demo

A five-letter word scramble quiz.

- Unscramble a random English word
- Ask for a clue when you need one
- Three guesses per round

## Local development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy

Push to `main` on GitHub. Vercel builds and deploys automatically once the repo is connected.
